# WEEK-1 — JavaScript Fundamentals

This week covers the basics of JavaScript programming including variables, conditionals, loops, functions, and array manipulation.

## 📁 Structure

```
WEEK-1/
├── 18-02-2026/          # Day 1 – Arrays, Loops & Conditionals
│   ├── ArraySum.js       # Sum of array elements using a function
│   ├── LargestAmong2Nos.js  # Compare two numbers (if-else)
│   ├── LargestAmong3Nos.js  # Largest of three using ternary operator
│   ├── MaxOf3Nos.js      # Max of three using a function
│   ├── MinOfArray.js     # Find minimum element in an array
│   ├── SearchArray.js    # Linear search in an array
│   └── SumOfArray.js     # Sum of array elements (inline)
└── 19-02-2026/          # Day 2 – Array of Objects & splice()
    └── ASSIGNMENT.JS     # Employee data manipulation with splice()
```

## 🧠 Concepts Covered

| Concept | Files |
|---|---|
| Variables & Data Types | All files |
| if-else / Ternary Operator | `LargestAmong2Nos.js`, `LargestAmong3Nos.js` |
| Functions | `ArraySum.js`, `MaxOf3Nos.js`, `SearchArray.js` |
| Loops (for) | `ArraySum.js`, `MinOfArray.js`, `SumOfArray.js`, `SearchArray.js` |
| Arrays of Objects | `ASSIGNMENT.JS` |
| Array.splice() | `ASSIGNMENT.JS` |
