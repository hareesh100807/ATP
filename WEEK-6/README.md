# WEEK-6 — React Forms & Mini Full-Stack App

This week covers React forms with validation using `react-hook-form`, and a full-stack mini application with separate backend and frontend.

## 📁 Structure

```
WEEK-6/
├── Form1.jsx          # User registration form with react-hook-form validation
└── Mini-app/          # Full-stack mini application
    ├── Backend/       # Express.js backend
    ├── Frontend/      # React frontend
    ├── package.json   # Project dependencies
    └── package-lock.json
```

## 🧠 Concepts Covered

| Concept | Files |
|---|---|
| react-hook-form | `Form1.jsx` |
| Form Validation (required, custom) | `Form1.jsx` |
| useState Hook | `Form1.jsx` |
| Rendering Lists in Tables | `Form1.jsx` |
| Full-Stack Architecture | `Mini-app/` |

## ▶️ How to Run

```bash
# Form component — import into a React project
# Mini-app
cd WEEK-6/Mini-app
npm install
npm start
```
