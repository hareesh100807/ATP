# WEEK-5 — CSS Flexbox, HTML Tables & React Components

This week covers CSS layout with Flexbox, HTML table structures, and introductory React components (Navbar, Footer, Product cards, User lists).

## 📁 Structure

```
WEEK-5/
├── FLEX/                # Flexbox layout exercise
│   ├── index.html       # Employee card layout with navbar
│   └── style.css        # Flexbox styling, card shadows, bg image
├── TABLES/              # HTML table exercises
│   ├── 1.html           # Multiple table layouts (colspan, rowspan, scroll)
│   └── style.css        # Table styling with borders and centered text
└── REACT/               # React component exercises
    ├── Navbar.jsx        # Navigation bar component
    ├── Footer.jsx        # Footer with location & contact
    ├── Product.jsx       # Product card with props
    ├── User.jsx          # User card (avatar, name, email)
    └── Userlist.jsx      # User list container with map()
```

## 🧠 Concepts Covered

| Concept | Files |
|---|---|
| CSS Flexbox (justify-content, gap) | `FLEX/style.css` |
| Box Shadow & Border Radius | `FLEX/style.css` |
| Background Images | `FLEX/style.css` |
| HTML Tables (colspan, rowspan) | `TABLES/1.html` |
| React Functional Components | All `REACT/*.jsx` |
| Props & Destructuring | `Product.jsx`, `User.jsx` |
| Array.map() for rendering | `Userlist.jsx` |
| Tailwind CSS (utility classes) | All `REACT/*.jsx` |
